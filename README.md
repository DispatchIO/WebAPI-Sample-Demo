# Sample projects for Dispatch Web SDK API 3.x

## 简介

此仓库包含基于 Dispatch Web SDK API 3.x 的示例项目。

## demo示例

使用jQuery和Bootstrap技术编写。

### 如何运行示例项目

#### 前提条件

- 示例部署请务必使用HTTPS协议或者localhost，并使用SDK支持的浏览器运行示例。
- 建议使用Google Chrome或Firefox浏览器。

#### 运行步骤

1. 使用SDK支持的浏览器打开demo/index.html，首先选择"用户登录"示例项目。
2. 在示例项目页面上，输入用户名和密码，登录系统成功获得access_token值。
 - 获得的token值，供调用接口鉴权使用。
2. 再选择一个示例项目进行测试。
